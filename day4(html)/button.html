button tag is used to create a clickable buttons button tag can be a type ->button: A generic button (default if the attribute is not specified). ->submit: Submits the form data to the server. ->reset: Resets the form data to its initial values. ->onclick: JavaScript code to run when the button is clicked. ->disabled: Disables the button, preventing it from being clicked. creating a button using input tag-we can create a button using input tag for this in input tag we have to use type submit and values attribute to show text on button
