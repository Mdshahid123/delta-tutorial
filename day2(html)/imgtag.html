Syntax: The tag is used to embed an image into an HTML document. It does not have a closing tag and is typically written as . Attributes: src: Specifies the path or URL to the image file. It can be a relative path (to the current HTML file) or an absolute URL. alt: Provides alternative text for the image. This text is displayed if the image fails to load or for users who use screen readers. It should describe the content or purpose of the image. width and height: Specify the dimensions (in pixels) of the displayed image. title: Provides additional information about the image, typically displayed as a tooltip when the user hovers over the image.
