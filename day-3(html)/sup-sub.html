Tag Definition: The tag is used to define subscript text, which appears slightly below the baseline. Use Case: Commonly used in chemical formulas, mathematical expressions, and other scenarios where subscript notation is required. example:

Water is H2O.

Tag Definition: The tag is used to define superscript text, which appears slightly above the baseline. Use Case: Commonly used for exponents in mathematical expressions, footnotes, and other scenarios where superscript notation is required. Example

Einstein's famous equation is E = mc2.
